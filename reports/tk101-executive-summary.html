이사님 요청 기능 — 은행 내역 + 세금계산서 자동 매칭

💡 왜 필요한가?

비유하자면, 지금은 40명이 각자 반복 업무를 수기로 처리하고 있습니다. AI 자동화 플랫폼은 회사 안에 "똑똑한 공용 비서"를 한 명 들이는 것과 같습니다. 모든 직원이 이 비서에게 업무를 맡길 수 있고, 회사는 그 사용 내역을 한눈에 관리할 수 있습니다.

현재 문제

  • AI 도구(ChatGPT, Claude 등)를 각자 개인 계정으로 쓰면 데이터 보안 문제가 생기고, 회사 차원에서 관리가 안 됨
  • 전 직원 유료 구독 시 월 수백만원 비용 발생
  • 업무 데이터가 회사 밖으로 유출될 위험
  • 반복 업무를 AI로 맡길 표준화된 방법이 없음

이 플랫폼이 해결하는 것

  • ✅ 회사 전용 AI 시스템 — 데이터는 회사 서버에만 저장
  • ✅ 비용 절감 — 개인 구독 대비 월 최대 90% 절감 (API 공용 사용)
  • ✅ 권한 관리 — 직급/부서별로 접근 가능한 데이터 범위 제한
  • ✅ 회계 자동화 — 은행 내역 정리, 세금계산서 매칭 등 반복 업무 자동화
  • ✅ 확장성 — 팀별로 필요한 기능을 추가로 개발 가능

🏗️ 어떻게 만들고 있나?

3단계 구조

🖥️ 1층: 직원 화면

직원이 보는 웹사이트. 로그인 후 AI와 대화하거나 원하는 업무를 요청할 수 있는 화면.

🧠 2층: AI 처리

요청을 받아 Claude(주) / OpenAI / Gemini 같은 AI 엔진에 연결. 답변을 돌려보냄.

💾 3층: 데이터 보관

회사 서버에 모든 데이터를 저장. 사용자 정보, 대화 이력, 업무 결과물을 안전하게 관리.

🔐 공통: 권한 관리

직급(관리자/팀장/일반)과 부서별로 볼 수 있는 데이터와 사용 가능한 기능을 구분.

어떤 업무가 자동화되나?

부서자동화할 수 있는 업무 (예시)
회계팀은행 거래내역 정리, 세금계산서 매칭, 계좌간 이체 자동 감지, 월별 리포트 자동 생성
마케팅팀체험단/인플루언서 DM 초안 작성, 가이드라인 준수 검토, 월간 성과 보고서, 전월 대비 인사이트 도출
디자인팀고객 대화 기반 디자인/영상 브리프 자동 작성, 요청사항 정리
신사업팀물류/클라우드 업무 관련 분석 및 보고서 작성 보조
공통이메일 초안, 회의록 요약, 문서 요약, 업무 질문 답변

🔒 보안 & 안전성

💬 "회사 데이터가 외부로 나가지 않나요?"

회사 서버(Tencent 클라우드, 한국 리전 선택 가능)에 모든 데이터를 저장합니다. AI에게 질문할 때만 텍스트가 전송되며, 민감 정보는 AI에 보내지 않도록 설계합니다. 데이터는 회사 자산으로 관리됩니다.

🔑 "누구나 모든 정보를 볼 수 있나요?"

아닙니다. 직급(관리자/팀장/일반 직원)과 부서별로 접근 권한이 다릅니다. 예를 들어 회계팀 일반 직원은 회계 데이터만, 관리자만 전체 시스템 설정에 접근 가능합니다.

💰 "비용이 통제되나요?"

부서별로 월 AI 사용량을 추적하고 상한선을 설정할 수 있습니다. 간단한 질문은 저렴한 AI 모델(Haiku)로, 복잡한 분석만 비싼 모델(Sonnet/Opus)로 자동 분류되어 비용을 절감합니다.

👤 "담당자가 퇴사하면 어떻게 되나요?"

처음부터 회사 명의의 GitHub Organization과 회사 명의 API 키를 사용합니다. 인수인계 시 소유권만 이전하면 전체 시스템을 넘길 수 있도록 설계되었습니다.
