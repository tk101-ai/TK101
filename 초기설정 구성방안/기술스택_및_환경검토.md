## 기술 스택 추천

### 추천: Python(FastAPI) + Next.js(React) + PostgreSQL

#### 백엔드: FastAPI (Python)
- AI 라이브러리 생태계 압도적 (Anthropic SDK, pandas, LangChain 등)
- Claude Code가 Python 코드 생성 품질이 가장 높음
- 타입 힌트 + Pydantic으로 자체 문서화 → 1인 유지보수에 유리
- 회계 자동화에 필요한 데이터 처리 라이브러리 풍부

#### 프론트엔드: Next.js (React)
- 채팅 UI + 백오피스를 하나의 프로젝트로 관리 가능
- SSR, 미들웨어, 인증 등 풀스택 기능 내장
- API Routes로 메신저 웹훅 프록시 간편

#### DB: PostgreSQL
- JSON 타입으로 AI 응답/프롬프트 이력 저장에 적합
- pgvector 확장으로 향후 RAG/시맨틱 검색 대응
- 회계 데이터의 트랜잭션 안정성 우수

#### 배포 구성
- Tencent CVM에 Docker Compose로 FastAPI + PostgreSQL + Redis 운영
- Next.js는 같은 CVM 또는 별도 배포
- GitHub Actions로 CI/CD 자동화

| React/Next.js 프론트 | typescript-reviewer + frontend-patterns + nextjs-turbopack | 85% | 거의 완비 |
