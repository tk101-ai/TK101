1순위. 체험단/인플루언서 DM 초안 생성 (#14)
================================================================

  왜 1순위?
  - 가장 간단. 10분이면 결과물 확인 가능
  - 성공하면 바로 실무 적용 가능
  - "AI가 이런걸 해준다"는 걸 보여주기 가장 좋은 예시

  테스트에 필요한 것:
  ┌──────────────────────────────────────────────────┐
  │ 1) 인플루언서/체험단 리스트 3~5명               │
  │    (이름, 채널, 팔로워 수, 특징 등)              │
  │ 2) 브랜드/캠페인 간단 설명                      │
  │ 3) DM 톤 지정 (친근/비즈니스/캐주얼 등)         │
  └──────────────────────────────────────────────────┘

  없으면? → 가상 데이터로 먼저 돌려볼 수 있음

  예상 결과물:
  - 대상별 맞춤 DM 초안 3~5개
  - 톤 변형 버전 (친근/공식 등)


================================================================
2순위. 가이드라인 준수 여부 검토 (#12)
================================================================

  왜 2순위?
  - 룰만 정해주면 자동 체크. 반복 사용 가치 높음
  - 사람이 놓치기 쉬운 부분을 AI가 잡아줌

  테스트에 필요한 것:
  ┌──────────────────────────────────────────────────┐
  │ 1) 가이드라인 룰                                 │
  │    - 금지 표현 목록                              │
  │    - 필수 포함 문구                              │
  │    - 해시태그/브랜드명 표기 규칙                 │
  │ 2) 검토 대상 콘텐츠 텍스트 1~2개                │
  └──────────────────────────────────────────────────┘

  없으면? → 샘플 가이드라인 + 샘플 콘텐츠로 시연 가능

  예상 결과물:
  - 항목별 Pass/Fail 체크 결과
  - 수정 필요 포인트 + 수정 제안


================================================================
3순위. 고객 대화 기반 브리프 작성 (#6)
================================================================

  왜 3순위?
  - 회의록/카톡 텍스트만 붙여넣으면 브리프 초안이 나옴
  - 디자이너/영상팀에 전달할 수 있는 실질적 결과물

  테스트에 필요한 것:
  ┌──────────────────────────────────────────────────┐
  │ 1) 고객과의 대화 내용                            │
  │    (카톡 캡처 텍스트, 메일, 회의록 등)           │
  │ 2) 결과물 유형 지정                              │
  │    (디자인 브리프 / 영상 브리프)                  │
  └──────────────────────────────────────────────────┘

  없으면? → 가상 고객 대화로 시연 가능

  예상 결과물:
  - 요구사항 정리
  - 누락된 질문 목록
  - 디자이너/영상팀용 브리프 초안


================================================================
4순위. 전월 대비 운영 성과 인사이트 (#10)
================================================================

  왜 4순위?
  - 숫자 데이터만 주면 분석+문장화+액션아이템까지 출력
  - 매월 반복 업무라 자동화 효과 큼

  테스트에 필요한 것:
  ┌──────────────────────────────────────────────────┐
  │ 1) 전월/당월 성과 데이터                         │
  │    (팔로워, 도달, 참여율, 전환 등)               │
  │ 2) 채널/매체 구분                                │
  │ 3) 보고 양식 (있으면)                            │
  └──────────────────────────────────────────────────┘

  없으면? → 샘플 마케팅 성과 데이터로 시연 가능

  예상 결과물:
  - 증감률 계산 + 원인 가설
  - 보고용 문장
  - 액션 아이템 제안


================================================================
5순위. 월간 보고서 초안 (#1)
================================================================

  왜 5순위?
  - 위 4개 테스트 경험이 쌓인 후 진행하면 품질 올라감
  - 데이터+템플릿 필요해서 준비 시간이 조금 더 필요

  테스트에 필요한 것:
  ┌──────────────────────────────────────────────────┐
  │ 1) 보고서에 들어갈 데이터                        │
  │ 2) 기존 보고서 템플릿 또는 양식                  │
  │ 3) 보고 대상 (대표/팀장/클라이언트 등)           │
  └──────────────────────────────────────────────────┘

  예상 결과물:
  - 템플릿에 맞춘 보고서 초안
  - 요약 문장 + 인사이트


================================================================
■ 테스트 진행 방식
================================================================

  방법 A) 실제 데이터로 테스트
  ────────────────────────────
  위 항목별 "필요한 것"을 준비해서 전달
  → 실제 업무 결과물이 바로 나옴
  → 가장 좋은 방법

  방법 B) 가상 데이터로 시연
  ────────────────────────────
  데이터 준비가 안 되면
  → "샘플로 한번 돌려봐" 하면 가상 데이터로 시연
  → AI가 어떤 결과를 내는지 미리 확인 가능
  → 이후 실제 데이터로 교체하면 됨
