## 1. 보안 정책 — 어디까지 어떻게 잠글 것인가

### 배경
사이트 자체 보안 수준을 어떻게 끌어올릴지 정해야 마음 놓고 운영이 가능한 상태가 된다.

### 논의 포인트

| 항목 | 현재 상황 / 고민 |
|------|------------------|
| **HTTPS 도입** | 도입 시 인증서 비용 부담. 미도입 시 사이트 노출 → API 무단 사용 리스크 발생 |
| **가입 방식** | 가입승인제로 운영. 중간중간 권한 설정·부여는 관리자가 언제든 조정 가능해야 함 |
| **사용자 소속 구조** | 사용자별 **중복 소속** 지원 필요 (관리자는 여러 부서 내용을 알아야 하는 경우 있음) |
| **접속 제어** | IP별 제어 도입 여부 / 재택근무자의 접근 권한을 어떻게 부여할지 |

### 운영 원칙
- 진행하면서 조정해 나가는 방식.
- **최종 목표**: 보안이 어느 정도 갖춰져야 마음 놓고 운영이 가능한 수준에 도달.

## 2. 텐센트 AI API 활용 방향

### 배경
텐센트 AI API를 통해 사용이 간편해진 상황. 기존/신규 모듈에 어떤 방식으로 녹여낼지 결정이 필요하다.

### 논의 포인트

| 항목 | 검토 내용 |
|------|-----------|
| **자체 서비스 연동** | 설정만 잘하면 API로 우리 서비스에서 언제든 AI 작업 가능 |
| **외부 메신저 연동** | 카카오/슬랙/팀즈 등 메신저 연동 가능하게 |
| **Claude API 대체 범위** | 기존 Claude API로 동작 중인 작업을 텐센트로 어디까지 대체할 것인가 |
| **기반 구조 수정** | 현재 진행 중인 프로젝트 기반을 수정해야 하는지 검토 |
| **AI 권한 위임 수준** | AI에게 어디까지 권한을 줘서 작업을 시킬 것인가 |
| **MCP 서버 구축** | 구글·슬랙·기타 외부 서비스 연동을 위한 MCP 서버 필요 여부 |
| **기존 모듈 AI 적용** | 이미 개발된 모듈에 AI 기능을 어떻게 얹을지 |
| **신규 모듈 설계** | 앞으로 추가될 모듈은 어떤 방식으로 AI를 녹여 개발할지 |

### 핵심 인식
> 비교적 **AI 사용에 대한 제약이 풀린 상태**라서 다양한 형태로 활용이 가능함.

## 3. 비용 — 가장 민감하고 답을 계속 찾아야 하는 주제

### 배경
모든 사용자가 API를 사용하는 구조가 되면 비용이 폭증할 수 있다. 근본적인 절감 방안을 모색해야 한다.

### 논의 포인트

#### 3-1. 구독 vs API 비교
- 기존 ChatGPT / Gemini 구독제는 비용이 많이 안 나옴.
- 솔직한 평가: **대화형은 ChatGPT·Gemini가 더 좋고 사용에 익숙함**.
- 그렇다면 구독 해제 + API 전환이 정말 절감이 되는가?
- **차별점**을 어디서 만들고 어떤 사용감을 줄 것인가가 핵심 질문.

#### 3-2. 비용 시뮬레이션
| 사용 패턴 | 예상 비용 |
|-----------|-----------|
| 사용자 1인 한도 | $10/일 |
| 일 20명 사용 시 (텍스트만) | $200/일 |
| 이미지·영상 생성 포함 | 토큰 비용 추가 상승 |

#### 3-3. 절감 방안 검토
- **오케스트레이션** — 가능은 하지만 근본적 해결책은 아님.
- **유인원 모드(자동 결과 모드)** — 결과만 받고 싶을 때 사용 가능하게 유도?
- **사용 시나리오별 분기** — 대화형으로 충분한 작업은 대화형으로, 자동화가 필요한 작업만 API로.

### 정리
가장 민감한 부분. 답이 하나로 정해지지 않으며 **계속해서 방법을 모색해야 함**.

## 4. 당장 떠오르는 추가 고려 사항

### 4-1. AI Playground ↔ 문서 자동작성 연동
- Playground로 문서 내용을 추출
- 추출한 내용을 문서 자동작성 모듈과 연동하여 한 흐름으로 사용 가능하게

### 4-2. 클라우드 정산 모듈화
- 기존에 수작업으로 처리하던 정산 작업을 모듈로 개발
- 작업 자체는 오래 걸리지 않을 것으로 예상

### 4-3. 박이사님 요청 — 거래내역 자동 분석
- 인증서를 등록해두면 AI가 자동으로 거래내역을 조회
- 조회 결과를 모듈에 적용하여 데이터 분석까지 자동화

### 4-4. AI 사용 강제 지양
- 당장 모든 작업을 AI로 시키지 말 것.
- **대화형으로 해결 가능한 일은 대화형으로 사용하도록 유도**.

## 회의 진행 시 권장 순서

1. **보안 (1번)** — 의사결정이 가장 시급. 운영 안정성 기반.

3. **텐센트 AI 활용 (2번)** — 1·3 결정 후 구체 적용 범위 논의.
4. **추가 고려 사항 (4번)** — 후속 과제 목록으로 정리.
